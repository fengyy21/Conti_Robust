# CT_Robust_Dispatch_Data

本仓库提供论文 **“新能源大基地连续时间域鲁棒调度方法”** 算例分析中使用的基础数据。

## 文件说明

* `ren_1h.mat`：1 h 时间分辨率下的新能源出力数据，用于连续时间鲁棒调度算法的基本运行。
* `ren_5min.mat`：5 min 时间分辨率下的新能源出力数据，用于高时间分辨率回放验证和算法基本运行。
* `ren_15min.mat`：15 min 时间分辨率下的新能源出力数据，用于对比算例构建。
* `dem_1h.mat`：1 h 时间分辨率下的参考功率轨迹数据，用于连续时间鲁棒调度算法的基本运行。
* `dem_5min.mat`：5 min 时间分辨率下的参考功率轨迹数据，用于高时间分辨率回放验证和算法基本运行。
* `dem_15min.mat`：15 min 时间分辨率下的参考功率轨迹数据，用于对比算例构建。

## 数据说明

新能源出力数据主要用于典型场景构造、Lipschitz 连续不确定集边界生成以及实时轨迹回放验证。其中，1 h 和 5 min 数据用于所提连续时间鲁棒调度算法的基本运行与验证，15 min 数据用于构建对比算例。参考功率轨迹数据用于构造调度目标，并用于不同时间分辨率下调度结果的对比分析。

## 单位说明

* 功率单位：GW
* 时间分辨率：1 h、15 min、5 min
* 数据格式：MATLAB `.mat` 文件

## 使用方法

可在 MATLAB 中直接读取数据文件，例如：

```matlab
load('ren_5min.mat')
load('dem_5min.mat')
```
